# Pearson's Chi Squared Test

## Author

Nathan Englehart (Summer 2022)

## Usage

To run this program on any unix-based system

```bash
git clone https://github.com/nathanenglehart/chi-squared-test
cd chi-squared-test
gcc -lm chi-squared.c -o chi-squared
```
